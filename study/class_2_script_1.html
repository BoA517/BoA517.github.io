마크다운(Markdown)

format된 텍스트는 보기에 좋습니다. 그러나 마우스클릭을 여러번 하는식으로는 큰프로젝트를 감당하기 힘듭니다. 우리에게 필요한건 마우스 클릭없이 기호로 텍스트를 프로세스할수 있는 능력입니다.

마크다운은 마크업언어입니다. 마크업언어는 특별한 코딩문자를 혼합하여 텍스트 레이아웃과 간단한 스타일을 지정할수있습니다.

아래의 마크다운 코드를 Atom editor에서 day01폴더안에 sample.md 파일을 만들어 복붙해주세요. ‘markdown preview: toggle’를 커맨드 팔렛(command palette)[CMD + SHIFT + P]에서 찾으셔서 적용하시면 마크다운이 HTML document로 parsing되고 새로운 탭에 rendering이 됩니다.

# 이것은 헤더(Header)입니다.
## 이것은 작은 헤더(Header)입니다.
### 더 작은
###### 완전 작은

이것은 보통 글입니다. 문장(paragrph)을 쓸수있습니다!

*이것은 이탈릭체(italic)로 바꾸는 기호입니다.*

**이것은 굵은 서식 기호입니다.**

***이것은 이탈릭과 굵은 서식 기호입니다.***

~~이것은 글위에 취소선을 그을수 있는 기호입니다.~~

**Paragraph안에** *스타일이 적용된 텍스트*를 넣을 수도 있지요. 꽤 쓸만하죠?!

번호없는 목록 (unordered list);

* Item 1
* Item 2
* Item 3

아래처럼 써도 같은 목록이 됩니다,

+ Item 1
+ Item 2
+ Item 3

아래처럼 써도 되고요,

- Item 1
- Item 2
- Item 3

순서있는 목록을 쓰시려면,

1. Item 1
2. Item 2
3. Item 3

목록안에 목록을 넣으려면,

1. Item 1
2. Item 2
3. Item 3
    1. Sub-item 1
    2. Sub-item 2
4. Item 4
